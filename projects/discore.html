Discore August 2016 - Now

An unofficial .NET Core library for interacting with the Discord API.

The objective of Discore is to fully implement two parts of the Discord API: the HTTP API, and the real-time WebSocket API. It's designed for creating Discord bots, as well as any application that does not require real-time data.

You can check out Discore on GitHub here.

Why?

Back in August 2016, I had wanted to create my own Discord bot for personal use in C#. At the time, there were a couple existing .NET libraries for this, however (at the time) neither supported .NET Core. So, I decided to have a go at creating my own library! At first this project started out just for personal use, but I eventually put it up on GitHub for anyone to use, and a friend of mine showed me how to upload releases to NuGet to make it easier for developers to download.
